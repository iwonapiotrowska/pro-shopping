# Pro Shopping - shop like a pro!

## Setup
1. Download repo
2. Enter the project's directory:
    ```
    cd pro-shopping
    ```
3. Install dependencies:
    ```
    npm install
    ```
4. Run the application:
    ```
    npm run serve
    ```

## Testing
1. Run linter:
    ```
    npm run lint
    ```

## Tools
1. JavaScript
  * **Vue.js** as JS framework
  * **ESLint** as JS linter
  * **Vuex**
  * **Vue Router**

2. CSS
  * **Sass** preprocessor
  * **BEM** methodology

3. HTML
  * **Pug** preprocessor
